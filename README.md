# A Tutorial of NodeJS RESTful API
After a long time of not building API's, I've undertaken to go through the basics again.

## Versioning
This project was generated using versions:
    - node 25.2.1
    - npm 11.6.2

Install npm packages using: `npm i`

## How to use
Run locally using: `node server.js`
View `http://localhost:3000`
